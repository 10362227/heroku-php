# Docker image for Heroku PHP [![Build Status](https://travis-ci.com/robuust/heroku-php.svg?branch=master)](https://travis-ci.com/robuust/heroku-php)

Usage: `FROM robuust/heroku-php`

# Specifications

* Heroku 18
* Nginx
* PHP 7.2.x with Redis and Imagick
* Composer
* Node 10.x
* Yarn
* Chrome
* ChromeDriver
